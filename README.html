Description

Provides data sets and R Codes for E.R. Williams, C.E. Harwood and A.C. Matheson (2023). Experimental Design and Analysis for Tree Improvement, CSIRO Publishing https://www.publish.csiro.au/book/3145/.

   

Installation

The package can be installed from CRAN as follows:

install.packages("eda4treeR", dependencies = TRUE)

 

The development version can be installed from github as follows:

if (!require("remotes")) install.packages("remotes")
remotes::install_github("myaseen208/eda4treeR")
